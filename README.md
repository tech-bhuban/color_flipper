# Color Flipper

## What it does
Changes background color randomly with different formats.

## Features
- Random HEX colors
- Random RGB colors
- Shows current color code
- Smooth transitions

## How to use
- Click "Random HEX" for hexadecimal colors
- Click "Random RGB" for RGB colors
- See color code displayed

## Example
- HEX: #FF5733
- RGB: rgb(255, 87, 51) -->
